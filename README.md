<h1 align="left">Hey👋🏻 ¿Qué tal?</h1>

###

¡Hola! Soy Joshua Vargas, un Desarrollador de Software con 2 años de experiencia en la creación de aplicaciones web robustas y eficientes. Actualmente, he enfocado mi carrera en el ecosistema de Python, con el objetivo de construir soluciones escalables y explorar el potencial de los datos para generar insights valiosos.

Apasionado por resolver problemas complejos, mi misión es transformar ideas en software funcional y de alto rendimiento. Me encuentro cursando un Técnico Superior Universitario en Informática, lo que complementa mi experiencia práctica con una sólida base teórica.

Mi enfoque técnico incluye:

- Lenguaje Principal: Python.

- Frameworks Backend: Django (para el desarrollo completo y rápido de aplicaciones complejas), Flask (para soluciones ligeras y personalizadas) y FastAPI (para la creación de APIs de alto rendimiento).

- Web Scraping & Automatización: Extracción estratégica de datos de la web utilizando herramientas de Python como Scrapy, BeautifulSoup y Selenium para alimentar análisis y procesos de negocio.

- Bases de Datos: Diseño, gestión y optimización de bases de datos tanto relacionales (MySQL, PostgreSQL) como NoSQL (MongoDB), asegurando la integridad y el rendimiento del almacenamiento de datos.

- Interés en Datos: Dando mis primeros pasos en el análisis y procesamiento de datos con librerías como Pandas, con el objetivo de especializarme en la ingeniería y análisis de datos.

Estoy comprometido con el aprendizaje continuo y la adopción de mejores prácticas. Mis áreas de interés actual son las arquitecturas de microservicios, la optimización de consultas y el ciclo de vida completo de los datos.
