<h1 align="left">Hey👋🏻 ¿Qué tal?</h1>

###

¡Hola! Soy Joshua Vargas 👋🏻

Software Developer con más de 3 años de experiencia, especializado en el desarrollo backend y el manejo de datos. Mi enfoque principal está en el ecosistema Python, construyendo APIs sólidas con Django, Flask y FastAPI, y diseñando arquitecturas escalables orientadas a la eficiencia y seguridad.

Tengo experiencia trabajando con bases de datos SQL y NoSQL, aplicando buenas prácticas en modelado, optimización de consultas y diseño de pipelines de datos. También me interesa el análisis y procesamiento de información, integrando herramientas de Python para Data Engineering y automatización.

Me apasiona escribir código limpio, aplicar principios de arquitectura de software y contribuir en equipos donde la calidad, la escalabilidad y el valor real del producto sean prioridad.

Actualmente finalizo el último semestre del TSU en Informática, mientras sigo perfeccionando mis habilidades técnicas y ampliando mi especialización en el área de backend y datos.
