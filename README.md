<h1 align="left">Hey👋🏻 ¿Qué tal?</h1>

###

¡Hola! Soy Joshua Vargas 👋🏻

Software Developer con más de 3 años de experiencia, especializado en el ecosistema Python. Mi enfoque principal es el desarrollo de arquitecturas backend escalables y confiables, con el objetivo de especializarme en Ingeniería de Datos para ampliar mi impacto en soluciones de procesamiento de información.

⚙️ Backend: Desarrollo APIs de alto rendimiento con FastAPI y Django, aplicando principios de escalabilidad, buenas prácticas de arquitectura y código limpio.

💾 Bases de Datos: Experiencia en el modelado y administración de bases de datos relacionales (PostgreSQL) y no relacionales (MongoDB), cuidando la consistencia, el rendimiento y la estructura.

🤖 Automatización: Implemento soluciones de Web Scraping y procesos automatizados para la extracción y procesamiento eficiente de información.

🎓 Formación: TSU en Informática, con fundamentos sólidos en desarrollo de software y una mentalidad orientada a la mejora continua y la resolución de problemas técnicos.
