<h1 align="left">Hey👋🏻 ¿Qué tal?</h1>

###

¡Hola! Soy Joshua Vargas 👋🏻

Software Developer con más de 3 años de experiencia, especializado en el desarrollo de aplicaciones web y APIs utilizando principalmente el ecosistema JavaScript/TypeScript.

Trabajo con Node.js, Express.js y NestJS para construir APIs eficientes, seguras y escalables, aplicando principios de arquitectura limpia, buenas prácticas y diseño orientado a la mantenibilidad.

En el frontend, desarrollo interfaces modernas y dinámicas con Angular, enfocándome en la creación de experiencias de usuario fluidas, componentes reutilizables y una comunicación robusta con el backend.

Tengo experiencia con PostgreSQL y MongoDB, abarcando desde el modelado y optimización de datos hasta la implementación de consultas complejas. También he desarrollado proyectos de Web Scraping, integrando la extracción y procesamiento de información en flujos eficientes.

Apasionado por el código limpio, el diseño eficiente de sistemas y la resolución de problemas mediante la programación, disfruto colaborar en equipos que valoren la comunicación clara, las buenas prácticas y la mejora continua.

Actualmente curso el último semestre del TSU en Informática y continúo fortaleciendo mi perfil técnico en el área de desarrollo web full-stack y backend moderno, ampliando mis capacidades para enfrentar nuevos retos profesionales.
