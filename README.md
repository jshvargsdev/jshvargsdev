<h1 align="left">Hey👋🏻 ¿Qué tal?</h1>

###

¡Hola! Soy Joshua Vargas 👋🏻

Software Developer con +3 años de experiencia en el desarrollo de aplicaciones web, especializado en el ecosistema JavaScript / TypeScript.

En el frontend, desarrollo interfaces modernas, dinámicas e intuitivas con Vue.js y Nuxt, priorizando la experiencia de usuario y la modularidad de componentes.

En el backend, trabajo con Node.js, Express.js y NestJS para construir APIs eficientes, seguras y escalables, aplicando principios de arquitectura limpia, buenas prácticas y diseño orientado a la mantenibilidad.

Manejo PostgreSQL y MongoDB, para el modelado y la optimización de datos. Además, he desarrollado proyectos de Web Scraping, integrando la extracción y procesamiento de información en flujos automatizados y eficientes.

Apasionado por el código limpio, el diseño de sistemas bien estructurados y la resolución de problemas mediante la programación, disfruto colaborar en equipos que valoran la comunicación clara, las buenas prácticas y la mejora continua.

Actualmente curso el último semestre de un TSU en Informática, fortaleciendo mis habilidades en el desarrollo de software y ampliando mi capacidad para asumir nuevos retos profesionales.
